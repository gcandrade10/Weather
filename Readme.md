# Weather

Sample app using [MetaWeather Api](https://www.metaweather.com/api/)

We are using:

* [AppIntro](https://github.com/AppIntro/AppIntro) to display the Intro for new users
* Koin for DI
* Viewbinding
* Navigation
* Kotlin Coroutines
* [OkHttpProfiler](https://github.com/itkacher/OkHttpProfiler)
* Retrofit
